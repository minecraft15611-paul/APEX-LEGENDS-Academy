# APEX-LEGENDS-Academy

這是一個以知名遊戲《Apex Legends (英雄聯盟)》為主題開發的單頁式應用程式 (SPA)。
本專案旨在練習原生 JavaScript (Vanilla JS) 的 DOM 操作、非同步資料處理以及響應式網頁設計。

## 🛠 使用技術
- **前端開發**: 
  - 原生 JavaScript (ES6+) - 核心邏輯處理
  - HTML5 & CSS3 - 結構與樣式
  - Bootstrap 5 - 快速建構響應式版面 (RWD)
- **後端輔助**: 
  - PHP / MySQL - 實作訪客計數器 (Visit Count)
- **圖表呈現**: 
  - Chart.js - 用於展示武器或角色數據分析

## ✨ 核心功能
- **響應式導覽**: 適應桌面、平板與手機端。
- **動態內容切換**: 使用 Vanilla JS 實作 SPA 效果，切換頁面不跳轉。
- **訪客統計功能**: 透過自製後端邏輯記錄並顯示頁面瀏覽次數。
- **武器/地圖資料庫**: 整合 Apex 遊戲數據，提供視覺化的資訊查詢。

## 📂 專案結構
- `pages/`: 存放各功能模組的 HTML 內容。
- `assets/`: 存放圖片、CSS 樣式與自訂 JS 腳本。
- `visit_count/`: 訪客統計功能的後端處理程式碼。
